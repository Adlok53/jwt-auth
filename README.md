###Api:
Login: POST  /api/auth/login
Registration: POST  /api/users
Table: GET    /api/auth/users
Logout: POST /api/auth/logout
Refresh: POST /api/auth/refresh
Me: GET   /api/auth/me

###Стек:
Laravel
CORS
Vue
JWT
REST Api
PostgreSQL

###Функционал:

1. Когда пользователь заходит на сайт

Если токена нет то его отправляет на страницу с авторизацией и регистрацией

Если токен есть то его отправляет на страницу с его персональными данными

2. Если введение данные на авторизацию и регистрацию

Верные, то сервер отправляет токен. Клиент записывает этот токен в localstorage и отправляет пользователя на страницу с его персональными данными

Неверны, то клиент выводит ошибку

3. На странице с персональными данными

Выводятся данные пользователя

Можно перейти на страницу со всеми зарегистрированными пользователями

Можно выйти из аккаунта

4. Если пользователь переходит на страницу с зарегистрированными пользователями

Клиент проверяет валиден ли токен, если нет то перезаписывает его

Клиент выводит пользователей в таблицу

5. Если пользователь хочет зайти на

Несуществующий роут то его отправляет на страницу 404

Страницу list, personal без токена то его отправляет на страницу с авторизацией и регистрацией
